> ##### Description
“Let this ring bind us.” This ring of precious metals is actually two interlocking bands

The first is wider of the rings with a golden interior and a roughly textured blacked outer layer. On this outer edge, there is an inset thin strip of gold that holds the second band tightly in place. This second ring is a much simpler thin band of gold. This classical design has an inset pattern hidden within its flat edge, hinting at the mysteries it contains.  These pair of rings hold a soft pull to each other, yearning to be interlocked if separated.

- This Ring is formed by two interlocking bands that can be separated into 2 Rings forming Pair Mates.
- When worn by a single user, this ring functions as a Ring of Protection.

**Unity:** over the course of a rest, the user can let an additional creature attune to the item, splitting the two rings between both users. When attuned in this way, both rings gain the following functions while within 60ft of each other. 
- The users gain a +1 AC to their Armor Class

**Mine is Yours:** As a Bonus Action, the user is able to Teleport swapping places with other user if they are willing. Each ring can use this property once, and once used, that ring can’t use this property again until the next Dawn.

**Support Unbound:** As a Reaction when the user makes a Saving Throw, they can add the other attuned user’s Saving Throw Bonus to their roll as well. Each ring can use this property once, an once used that ring can’t use this property again until the next Dawn.

**Earnest Inspiration:** If the user chooses to spend Inspiration, the other user also gains Advantage on their next roll.

**Tethered:** While worn, the user always knows the direction of its Pair Mate as long as they are on the same plane. As a ring always pulls softly towards the other.

If a user unattunes from this item, that ring will appear interlocked with its Pair Mate after the next Long Rest.
